# Insect Classifier
Can classify 12 different types of harmful insects to the crops <br/>
The types are following: <br/>
1. Bees
2. Aphids
3. Armyworms
4. Cabbage Loopers
5. Corn Boreres
6. Corn Earwormes
7. Corn Rootwormes
8. Fruit Flies
9. Potato Beetles
10. Stink Bugs
11. Thrips
12. Tomato Hornworms
